#Interactive Valentine's Web Experience 💖

A beautifully crafted, single-page interactive web application designed with a romantic theme. This project features dynamic animations, a dual-timezone countdown, and integrated mapping.

[**Live Demo**](https://keerthika-kaliyan.github.io/valentine/)

#Features ✨

- **Interactive Map:** Uses `Leaflet.js` to visualize the distance between Maynooth, Ireland, and Hosur, India, with custom pulse markers.
- **Dynamic Animations:** - Falling emoji particles and rose petals.
  - An interactive "No" button that dodges the cursor.
  - A CSS-animated "Letter" with a wax seal opening effect.
- **Dual-Region Countdown:** Real-time countdown timers calculated for both IST (India) and GMT/IST (Ireland) time zones targeting October 22, 2026.
- **Audio Integration:** Interactive sound effects for button clicks and interactions.
- **Responsive Design:** Fully mobile-responsive layout using `Tailwind CSS`.

#Tech Stack 🛠️

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Maps:** [Leaflet.js](https://leafletjs.com/)
- **Fonts:** Google Fonts (Dancing Script, Quicksand, Special Elite)
